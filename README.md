## 💻 Projeto

API financeira estatica

## ✨ Tecnologias
- Node
- JavaScript
- Insomnia
- express
- nodemon

## 📚 Bibliotecas
- uuid
 
## 💻 Funcionalidades
- Criar conta
- Criar Depositos
- Mostrar Depositos
- Saque
- Listar operações
- Deletar conta

## 📱 Rotas

- POST → /account
- GET → /statement
- POST → /deposit
- POST → /withdraw
- GET → /statement/date
- PUT → /account
- GET → /account
- DELETE → /account
- GET → /balance


## 🚀 Como executar

1) Clonar o projeto do github:
    `git clone https://github.com/Luizrebelatto/FinApi.git`

2) Abrir terminal(CMD/TERMINAL ou VSCode/CMD) e acessar a pasta raiz do projeto (pelo terminal).
    

3) Instalar Dependências:
  execute `yarn`


4) Rodar Aplicação no http://localhost:3333:
    `yarn dev`
